技术

本周全球最大 开(同) 源(性) 技(交) 术(友)网站宣布可以无限提供私人代码仓库，协作者上限为3人。

git 中内置了一个快速定位 BUG 的命令，以方便用户在使用中定位未知的commit中对代码产生影响的bug。 使用 git bisect start HEAD oldcommit 来确定需要定位的commit区间。然后git会用二分法指向位于中间的commit 继续使用 git bisect good git则会取到中间到现在的 commit 中段的commit。在使用的过程中仅需要不断的观察程序是否在这一段的commit中仍然存在 BUG 即可（在找到BUG之后 git bisect reset 即可退出） – 你看人家连找BUG都帮你想好怎么解决了

经济

从二零一九年一月一号开始减税的政策正式开始执行，广大的纳税者可以根据自己的实际情况来进行填报税收信息。不过有很多没有正式租房合同的小伙伴在向房东获取基本信息的时候遇到了很大的阻力，一方减税领一方必然加税才能维持平衡。
