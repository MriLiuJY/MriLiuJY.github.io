谷歌高级搜索技巧

  • 使用 site:github.com 来固定搜索某一站点下的结果
  • 使用双引号 “搜索” 来精准匹配搜索的内容
  • 使用减号用来排除某一搜索结果 如: 搜索 -引擎
  • 使用星号来表示任意内容 如: 搜索*擎
  • 使用 inurl 来匹配在url 中的内容 如: inurl:web 会找出url中带有web的链接 (使用 allinurl 可以输入多个关键词 如: allinurl:web/fontend 其中/会被忽略)
  • 使用 intitle 来匹配 title包含关键字的结果（用法和上条一样）。
  • 使用 filetype 来制定搜索文件的类型 如: filetype:doc site:gov confidential 将会返回所有gov网站下.doc 文件和 包含 confidential 关键词的页面。
  • 使用 link 来搜索所有包含链接的页面 如: link:mriliujy.github.io/
  • 在两个数值之间搜索 如 year 1910.. 1920
